# Architecture Prompts

## ADR Request
```txt
This may require an architecture decision.
Do not implement yet.
Write a short ADR with context, goals, non-goals, options, tradeoffs, decision, security/data implications, rollback plan, and follow-up work.
```

## Dependency Decision
```txt
Before adding a dependency, write an ADR.
Compare using existing code/native APIs vs adding the dependency.
Include maintenance, security, bundle size, lock-in, and rollback considerations.
```

## API Design
```txt
Propose an API design before implementation.
Include request/response shape, error behavior, authorization model, backward compatibility, validation, tests, and migration path.
```
