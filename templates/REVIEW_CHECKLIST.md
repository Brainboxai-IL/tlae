# Review Checklist

## Correctness
- Does the code solve the stated problem?
- Are edge cases handled?
- Are errors handled safely?

## Scope
- Are changes focused?
- Any unrelated refactor?
- Any generated/vendor files changed?

## Security
- Auth/authorization correct?
- Input validation present?
- No secrets/PII leaked?
- Injection risks considered?

## Data
- Data integrity preserved?
- Migration safe?
- Backward compatible?

## Tests
- Meaningful tests added/updated?
- Regression covered?
- Manual testing documented?

## Production
- Logs/metrics/alerts needed?
- Feature flag needed?
- Rollback path clear?
